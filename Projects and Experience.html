Botbuster

Botbuster, my Final Year Project at SP, was a comprehensive text-analysis framework that could accept a variety of text sources such as websites, social media accounts and documents for analysis. Bot Buster would then make use of various detection engines to analyse the text and aggregrate their results, producing more accurate results than standalone detection engines.

Home Server Project

I repurposed an old laptop to work as a home server by installing ubuntu server on it. The server is currently not publicly accessible, however I have plans to set up a webserver to host this website a VPN server, plex media server and an minecraft server. The server currently runs services such as samba file storage, syncthing for phone backups and adguard home. I implemented all of these services using docker, so that I can learn and understand how containerization works.
